# Firebase-Service

## Requirement Specification

- SpringBoot 3.0.0
- Kafka
- Java 11

## Consume Topic Kafka

- `notification_firebase` = Consume Message Firebase Notification

## How's to Run Local

### MakeFile Commands

- `make run-docker-local`
- `make run-spring-boot`
- `make docker-push-latest`
- `make docker-push`
- `make maven-verify`
- `make list`
- `make docker-build-embed-tomcat`
